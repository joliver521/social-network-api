# Social Network API

## Description:

This project is an API set up as a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list.

## Installation

To run this project, install it locally using npm:

```
npm install
```

## Usage

After installing npm packages, the application will be invoked by using the following command:

```
npm start
```
